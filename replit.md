# AutonomOS Discovery Visualization

An interactive visual demonstration of the AutonomOS platform pipeline, showcasing the journey from Asset Discovery to Agent Execution.

## Features

- **Interactive Graph Visualization**: Detailed node-based view of the platform architecture.
- **Animated Data Flow**: Visual representation of data moving through discovery, mesh, unification, and agent stages.
- **Mock Data simulation**: Demonstrates how legacy and modern systems connect.
- **Frontend Only**: Built with React, React Flow, and Tailwind CSS. No backend dependencies.

## Running the Project

Click the **Run** button to start the development server.
The application will be available in the Webview.

## Commands

- `npm run dev:client`: Starts the frontend development server.
